说明

一、顶部导航栏说明

顶部导航栏伸缩的宽度需要根据实际情况进行调整

具体需要修改的代码在assets/css/index.css里面第100行至第150行

/*根据实际情况在此调整顶部宽度*/
@media (min-width: 1200px){
	.readboy-nav-contant{
		width: 1200px;
	}
}
@media (max-width: 1199px){
	.readboy-nav-contant{
		width: 100%;
	}
}

目前，在浏览器宽度大于1200px时，导航条内容区宽度不变，小于1200px时，导航条宽度为浏览器宽度

二、左右布局说明

具体的左右布局效果请参照DEMO

<div class="readboy-content">
    <div class="readboy-content-wrapper">
        <div class="readboy-content-wrapper-left">         
        </div>
        <div class="readboy-content-wrapper-right">
        </div>
    </div>
</div>

.readboy-content-wrapper类部分的div是居中的白底内容部分

.readboy-content-wrapper-left类是指左边部分，.readboy-content-wrapper-right类是指右边部分

左右两部分各占的大小也需要根据实际内容进行调整，相关的css代码在assets/css/index.css里面第200行至第300行

.readboy-content-wrapper{
	display: flex;
	margin: 0 auto;
	width:1100px;
	box-shadow: 0px 0px 8px 3px rgba(6,0,1,.04);
	background-color: white;
}
/*左小右大布局*/
.readboy-content-wrapper-left{
	flex: 0 0 260px;
	padding: 10px;
}
.readboy-content-wrapper-right{
	flex: 1;
	padding: 10px;
}
.readboy-content-wrapper-center{
	flex:1;
	padding: 20px
}

目前，居中部分白底内容区的宽度是1100px，具体宽度在.readboy-content-wrapper类里面修改。左边部分的宽度是260px，具体宽度在.readboy-content-wrapper-left类里面修改。右边部分的宽度是白底内容区剩下的空间。目前呈现的效果是左边小，右边大。如果要实现左边大，右边小的效果，只需要更换.readboy-content-wrapper-left和.readboy-content-wrapper-right的位置即可。

三、侧边菜单栏说明

侧边菜单栏的三个类.readboy-menu-1.readboy-menu-2.readboy-menu-3分别代替只有一级菜单、有二级菜单、有三级菜单的宽度。具体的宽度也可以自行设定。
